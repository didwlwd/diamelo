## 설치 및 실행 방법
```bash
git clone https://github.com/myuser/mytodoapp.git
cd mytodoapp

# 프론트
cd frontend
npm install
npm start

# 백엔드
cd backend
./gradlew bootRun

## 주요 기능
- 회원가입 및 로그인 (JWT 기반)
- 할일관리
- 일정관리

## 부가 기능

- 반응형 웹UI
